# 🌳 Decision Tree Classifier — Bank Marketing Dataset
## Prodigy InfoTech | Data Science Internship | Task 03

A machine learning project that builds a Decision Tree Classifier to predict
whether a bank customer will subscribe to a term deposit, based on their
demographic and behavioral data from a telemarketing campaign.

---

## 📌 Problem Statement
Banks run telemarketing campaigns to promote term deposits. Not every customer
subscribes — so the goal is to **predict in advance** who is likely to say YES,
helping the bank target the right people and save resources.

---

## 📦 Dataset
- **Name:** Bank Marketing Dataset
- **Source:** UCI Machine Learning Repository
- **Provider:** [Prodigy InfoTech Datasets](https://github.com/Prodigy-InfoTech/data-science-datasets/tree/main/Task%203)
- **Size:** ~45,000 rows | 17 features
- **Target:** `y` — Did the customer subscribe? (`yes` / `no`)
